# Application Tracker Example

Edit the real private tracker at `results/tracker/application_tracker.md`.

Simple workflow:
- Change `Status` from `NEW` to `APPLIED` after applying.
- Optionally fill in `Applied On`, `Follow Up On`, `Response`, and `Notes`.
- Rerun `main.py` so `apply_queue.md` reflects the tracker update.
